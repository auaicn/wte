# 음식 가계부 & 레시피 관리 앱

재료 가격 관리, 레시피, 가계부 및 식사 계획을 위한 Next.js 애플리케이션입니다.

## 주요 기능

- **재료 관리**: 음식 재료별 가격 추적 및 재고 관리
- **레시피**: 레시피 저장 및 관리, 재료 비용 계산
- **가계부**: 식비 지출 내역 추적 및 통계
- **식사 계획**: 캘린더를 통한 주간/월간 식사 스케줄링

## 기술 스택

- **프레임워크**: Next.js 15 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **린팅**: ESLint

## 시작하기

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

## 개발 스크립트

- `npm run dev` - 개발 서버 실행
- `npm run build` - 프로덕션 빌드
- `npm run start` - 프로덕션 서버 실행
- `npm run lint` - ESLint 실행
